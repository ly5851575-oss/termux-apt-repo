# سياسة الأمان

هذا المشروع يثبت ويشخّص أدوات أمنية، ولا ينفذ فحوصًا أو هجمات تلقائية.

- استخدم الأدوات فقط على أنظمة تملكها أو لديك تصريح مكتوب لاختبارها.
- لا ترفع مفاتيح SSH أو الرموز أو كلمات المرور أو ملفات الجلسات إلى المستودع.
- للإبلاغ عن ثغرة في السكربتات، استخدم GitHub Security Advisories إن كان متاحًا، ولا تنشر أسرارًا في Issue عام.
